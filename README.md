# Portfolio - Full-Stack Developer & AI Integration Expert

Professional portfolio website showcasing web development, AI integration, and automation projects.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: TailwindCSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Deployment**: Vercel

## Pages

- **Home**: Hero section + core expertise
- **Projects**: 3 real-world case studies
- **Skills**: Technical skills with proficiency levels
- **Contact**: Multiple contact methods + form

## Projects Featured

1. **Demand Discovery Engine**
   - Automated job collection and AI scoring
   - 235+ jobs collected
   - Tech: Python, PostgreSQL, LLM

2. **Browser Relay Anti-Scraping Solution**
   - Bypasses Cloudflare protection
   - Real browser session integration
   - Tech: Playwright, OpenClaw

3. **Bidding Proposal Generator**
   - AI-powered proposal generation
   - 15-30 second output
   - Tech: Python, Ollama
